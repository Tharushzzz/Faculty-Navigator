# 📍 Faculty Navigator

Faculty Navigator is a simple web-based navigation system that helps users find important locations inside a faculty area such as the Faculty Canteen and Computer Laboratory.

This project was created as part of an academic assignment to demonstrate how a navigation system can guide users through campus locations.

## 🚀 Live Demo
https://tharushzzz.github.io/Faculty-Navigator/

## 🎯 Features
- Easy navigation to faculty locations
- Step-by-step directions
- Simple and user-friendly interface
- Accessible through any web browser

## 📍 Destinations
- Faculty Canteen
- Computer Laboratory

## 🛠️ Technologies Used
- HTML
- CSS

## 📚 Purpose
This project demonstrates a basic campus navigation system that helps students and visitors easily locate important facilities within the faculty.
